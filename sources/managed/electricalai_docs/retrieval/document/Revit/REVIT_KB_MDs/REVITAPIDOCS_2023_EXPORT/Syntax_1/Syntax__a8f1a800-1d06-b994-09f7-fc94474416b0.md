#### Parameters

date
:   Type:  [System DateTime](http://msdn2.microsoft.com/en-us/library/03ybds8y)    
     The date for which to determine sunset time.

#### Return Value

The date and time. The value will be in Coordinated Universal Time (UTC).

# Remarks

The value returned is affected by the value of the  [UsesDST](34284848-ddf6-1fda-d1d2-43f8c2e2ad78.htm)  flag set for the current location. If this value is true, the sunset value will be adjusted for Daylight Savings Time, regardless of the value of the input date.
