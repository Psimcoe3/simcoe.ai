# Remarks

The referenced element may be any model element, including FamilyInstance or FormElement. The reference must be of type ElementReferenceType.REFERENCE\_TYPE\_SURFACE, and the surface must be of type  [Plane](6a6ee978-f114-558d-3c69-00d289aa855f.htm)  . The Reference's UVPoint property is ignored.
