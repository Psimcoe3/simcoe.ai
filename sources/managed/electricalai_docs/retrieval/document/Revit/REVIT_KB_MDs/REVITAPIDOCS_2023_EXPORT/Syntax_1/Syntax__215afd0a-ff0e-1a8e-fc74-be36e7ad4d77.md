Specifies the station at which the label will be placed, in Revit internal model units (standard Imperial feet). The station determines the location of the label's origin:  [Origin](df8b9dc6-9d36-ac2b-04cf-816d88f039b8.htm)  by setting it to the closest point on the alignment geometry which corresponds to this station.
